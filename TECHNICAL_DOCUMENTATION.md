
# Technical Documentation: Poultry Disease Detection System

This document provides a comprehensive technical overview of the Poultry Disease Detection System, a monorepo application consisting of a React frontend and an Express.js backend proxy that interfaces with a Python FastAPI machine learning service.

---

## 1. FILE STRUCTURE OVERVIEW

The project uses a monorepo structure managed by NX, separating the frontend (`apps/web`) and backend (`apps/api`).
